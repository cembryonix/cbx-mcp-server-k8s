# CBX MCP server for Kubernetes ![status](https://img.shields.io/badge/version-0.1.0--alpha-orange)

**An MCP server providing interface for CLIs for Kubernetes operations.** <br>
Part of the "Cembryonix Project" collection.

Allow LLMs to use Kubernetes management command line tools. 

> [!IMPORTANT]
> **Early Development Release**
>
> This is an initial release with basic functionality. We're actively working on:
> - Documentation
> - optimizing CLI execution functionality
> - optimizing the docker image size and build process
> 
> Expect frequent updates and potential breaking changes.
